Leaf is split across several repositories in the
[Utility-Muffin-Research-Kitchen](https://github.com/Utility-Muffin-Research-Kitchen)
organization. This page will map them out, including:

- **Leaf** — release artifacts and cross-repo orchestration.
- **Jawaka** — the launcher (daemon, launcher, menu, OSD).
- **Catastrophe** — the shared UI toolkit.
- **miniloong-launcher-switcher** — the boot hook, session supervisor, and
  device install/recovery mechanism.
- **mlp1-toolchain** — the cross-compile Docker image.
- **retroarch-builds** / **Cores-spruce** — RetroArch and libretro cores.
- App repos — the SSH server, file manager, and Fugazi.
