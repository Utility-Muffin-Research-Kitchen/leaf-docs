Because Leaf runs **on top of** the stock OS, recovery is straightforward — and
in most cases automatic.

This page will cover:

- **Automatic fallback** — how Leaf returns you to the stock interface if it
  fails to start cleanly several times in a row.
- **Recovery card** — using the `leaf-mlp1-recovery-<release_id>.zip` to restore
  the device to a clean stock state from an SD card.
- **Removing Leaf** — fully uninstalling and going back to stock for good.

In the meantime, see [Install Leaf](/leaf-docs/guide/install/) for how the
install/boot flow works.
